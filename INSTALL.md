# Установка Whisper Local

## 1. Установка приложения

Откройте `Whisper Local.dmg` и перенесите `Whisper Local.app` в `Applications`.

При первом запуске приложение скачает Python-библиотеки и модель Whisper `medium`. Это может занять несколько минут. Нужен интернет.

Данные приложения хранятся здесь:

```text
~/Library/Application Support/Whisper Local
```

Там будут виртуальное Python-окружение, модель, настройки и записи по умолчанию.

## 2. Разрешения macOS

При запросе разрешите доступ к микрофону. Если запись не стартует, проверьте вручную:

```text
System Settings -> Privacy & Security -> Microphone
```

В списке должно быть разрешение для `Whisper Local`.

## 3. Установка BlackHole

Если Homebrew уже установлен:

```bash
brew install --cask blackhole-2ch
```

После установки обязательно перезагрузите Mac.

Если Homebrew нет, скачайте BlackHole 2ch с сайта Existential Audio и установите пакет вручную:

```text
https://existential.audio/blackhole/
```

## 4. Настройка звука с наушниками

Откройте:

```text
Applications -> Utilities -> Audio MIDI Setup
```

Нажмите `+` в левом нижнем углу и выберите `Create Multi-Output Device`.

Внутри Multi-Output включите:

- `BlackHole 2ch`
- ваши наушники, например `WH-1000XM6`

Если не хотите слышать звук из динамиков MacBook, выключите `Динамики MacBook Pro`.

Для наушников обычно включите `Drift Correction`. `BlackHole 2ch` оставьте Primary/Master device.

Затем откройте звук macOS и выберите вывод:

```text
System Settings -> Sound -> Output -> Multi-Output Device
```

## 5. Настройка в Whisper Local

Для записи видео/браузера и микрофона:

- `Режим записи`: `Приходящий + микрофон`
- `Приходящий звук`: `BlackHole 2ch`
- `Микрофон`: ваш микрофон или микрофон гарнитуры
- `Язык распознавания`: `Русский`

Для записи только видео/браузера:

- `Режим записи`: `Только приходящий`
- `Приходящий звук`: `BlackHole 2ch`

Для записи только микрофона:

- `Режим записи`: `Только микрофон`
- `Микрофон`: нужное устройство

Важно: обычные наушники в списке входов не являются “звуком из наушников”. macOS не дает записывать выход напрямую. Поэтому для приходящего звука нужен `BlackHole 2ch`, `Loopback` или похожий виртуальный аудиовход.
